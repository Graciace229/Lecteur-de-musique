# Lecteur de musique

Petit projet Python : un lecteur de musique simple.

## Contenu
- `music_player.py` : script principal du lecteur.

## Installation
Si des dépendances sont nécessaires, ajoutez-les dans `requirements.txt` puis installez :

```bash
pip install -r requirements.txt
```

## Utilisation

Lancer le script :

```bash
python music_player.py
```
